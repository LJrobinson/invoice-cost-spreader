# Invoice Cost Spreader

Invoice Cost Spreader is a lightweight landed-cost calculator for cannabis operators.

It spreads invoice-level costs like freight, fees, and discounts across received inventory lines using explicit allocation methods.

## Product Doctrine

Do not blindly spread invoice costs per line.

Cannabis invoices often contain wildly different quantities per line. Spreading costs evenly per line can distort unit cost, COGS, and margin reporting.

This tool supports allocation by unit count or invoice value so operators can calculate more defensible landed costs.
